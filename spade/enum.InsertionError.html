Enum spade::InsertionError

source · 
pub enum InsertionError {
    TooSmall,
    TooLarge,
    NAN,
}
Expand description

The error type used for inserting elements into a triangulation.

Errors during insertion can only originate from an invalid vertex position. Vertices can be checked for validity by using crate::validate_vertex.

Variants§

§

TooSmall

A coordinate value was too small.

The absolute value of any inserted vertex coordinate must either be zero or greater greater than or equal to crate::MIN_ALLOWED_VALUE.

§

TooLarge

A coordinate value was too large.

The absolute value of any inserted vertex coordinate must be less than or equal to crate::MAX_ALLOWED_VALUE.

§

NAN

A coordinate value was NaN.
