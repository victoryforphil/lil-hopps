# Lil Interface

This is a basic UI for interacting with the lil-gcs.

This is used with Vite and npm.

## Building with Dev

```bash
cd lil-interface
npm run dev
```

## Building with docker

```bash
docker compose up -d --build
```

## Technologies

### Styling

Mostly uses Mantine UI and SCSS.

### Charting

Rechart is the go to charting library.
