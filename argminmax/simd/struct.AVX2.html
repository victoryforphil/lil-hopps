Expand description

AVX2 instruction set.

Type that implements the SIMDArgMinMax trait.

This struct implements the SIMDArgMinMax trait for the different data types and their datatype strategies.

Note that AVX is enough for f32 and f64, but we need AVX2 for all other data types.
